# Logo Instructions

## Add Your SafeTNet Logo

1. **Place your logo file here:**
   - File name: `safetnet-logo.png`
   - Full path: `src/assets/images/safetnet-logo.png`

2. **Logo Specifications:**
   - Format: PNG (recommended with transparent background)
   - Recommended size: 400x300px or similar aspect ratio
   - The logo will be automatically resized to fit the containers

3. **Where it will appear:**
   - ✅ Splash Screen (app startup)
   - ✅ Login Screen (replaces emoji logo)

## Note
If the logo file is missing, the app will show an error. Make sure to add the logo file before building the app.
